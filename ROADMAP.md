# Roadmap

## Vision

Make Pith the default open-source prompt distillation engine for developers.

## 0.2.0 (Near Term)

- Stabilize and document plugin API.
- Expand regression corpus for semantic preservation.
- Add benchmark comparison output for pull requests.
- Improve CLI UX for batch and stdin workflows.

## 0.3.0 (Mid Term)

- Add language profiles for PT/EN/ES/FR behavior tuning.
- Add optional safety modes (strict, balanced, aggressive).
- Publish benchmark dashboard artifacts from CI.

## 1.0.0 (Long Term)

- Lock public contract for `optimizeStable`.
- Publish migration guide from 0.x to 1.0.
- Maintain semantic versioning and release discipline.
