# Pith VS Code Extension

O **Pith** é um compressor e destilador de prompts Zero-G para modelos de IA. Esta é a versão oficial da extensão para o VS Code / Antigravity, trazendo o mesmo motor da extensão para Chrome diretamente para o seu editor de código.

## Funcionalidades

- **Optimize Selected Text:** Selecione qualquer texto e execute `Pith: Optimize Selected Text` no Command Palette (ou use o atalho `Cmd+Shift+L` / `Ctrl+Shift+L`) para aplicar a compressão e remoção de "gordura" usando o protocolo Zero-G.
- Funciona 100% offline — toda a compressão é feita localmente pelo `PithEngine`.

## Como usar (Instalação Manual)

1. Baixe ou gere o arquivo `pith-vscode-1.0.0.vsix`.
2. No VS Code / Antigravity, abra as Extensões (`Cmd+Shift+X`).
3. Clique no ícone de opções (...) no canto superior direito do painel de extensões.
4. Escolha `Install from VSIX...`
5. Selecione o arquivo `.vsix` deste projeto.

## Atalhos

- **Mac:** `Cmd+Shift+L`
- **Windows/Linux:** `Ctrl+Shift+L`
